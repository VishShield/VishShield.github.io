  1. Try Chrome browser (best WebGL support)
  2. Check if WebGL is enabled
  3. Clear browser cache (Ctrl+Shift+Delete)

Solution:
  1. Clear browser cache
  2. Check browser console (F12) for errors
  3. Make sure localStorage is enabled
